# 🗂️ Role-Based Task Management System

A responsive and user-focused task management web application built with **React** and **Tailwind CSS**, enabling efficient task assignment, tracking, and progress visualization through role-based dashboards.

---

## 🚀 Key Features

- **Role-Based Access Control**
  - Admins can assign, manage, and monitor tasks
  - Employees can track and update assigned tasks

- **Task Lifecycle Workflow**
  - Task states: **New**, **Active**, **Completed**, **Failed**
  - Real-time task counters for instant progress insights

- **Centralized State Handling**
  - Manages authentication, user data, and tasks seamlessly
  - Ensures consistent data flow across the application

- **Session Persistence**
  - User sessions and task data stored using browser local storage
  - Preserves state across page refreshes

- **Modern & Responsive UI**
  - Fully responsive dashboards styled with **Tailwind CSS**
  - Modular, reusable React components for scalability

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **State Management:** Centralized React state  
- **Storage:** Local Storage
